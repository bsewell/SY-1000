### 4. `src/floorBoard.cpp` — Signal chain column alignment fix (TWO root causes)

**Root cause 1 — flowStep mismatch (line 1679):**
```cpp
// BEFORE (int truncation):
const int flowStep = 55*ratio;  // 60 at ratio 1.1

// AFTER (matches ChainLayout):
const int flowStep = qRound(55.0*ratio);  // 61 at ratio 1.1
```
`ChainLayout::m_flowStep` used `qRound(55.0*ratio) = 61` but `update_structure`
used `int(55*ratio) = 60`. Balancers (positioned via `layout.colX()`) used 61px
columns while FX blocks advanced at 60px, causing cumulative 1px/column drift.

**Root cause 2 — compactSegmentAfterBalancer (lines 2244-2246):**
```cpp
// BEFORE (broke column alignment):
compactSegmentAfterBalancer(29, 31, false);
compactSegmentAfterBalancer(31, 33, false);
compactSegmentAfterBalancer(33, -1, true);

// AFTER (disabled — Boss layout uses strict column grid):
// compactSegmentAfterBalancer(29, 31, false);
// compactSegmentAfterBalancer(31, 33, false);
// compactSegmentAfterBalancer(33, -1, true);
```
Post-layout compaction shifted entire segments after each balancer to close gaps,
breaking cross-row column alignment. FX3/Comp were shifted -14px left of FX1/FX2.
Two Boss reference screenshots confirm strict column alignment with no compaction.

**Result — zero column error across all 34 chain positions:**
```
Cross-row column alignment:

  Bal1 = Amp: x=[438, 438]  ALIGNED
  Bal2 = NS: x=[560, 560]  ALIGNED
  All blocks: col err=0
```

## Boss Reference Layout Rules (from 2 screenshots)

1. **Strict column alignment** — blocks at the same column index across different
   rows share the exact same X position. No drift, no compaction shift.
2. **Uniform column width** — `flowStep = qRound(55.0 * ratio)` = 61px at 1.1x.
   Every column after C0 is one block-width (44px) + one gap (17px).
3. **C0 is wider** — instrument blocks are ~88px (2x the 44px FX block width).
4. **All rows start at C1** — after each split/row-reset, blocks begin at the
   same `firstFlowX` position (column 0 in the FX grid).
5. **Balancers at `max(inputA, inputB) + 1`** — positioned one column past the
   longer of the two input paths.
6. **Empty rows draw a wire** — rows with no FX blocks just show a connecting
   wire to the next balancer.
