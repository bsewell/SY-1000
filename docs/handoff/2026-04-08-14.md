### 3. `src/SysxIO.cpp` + `src/midiIO.cpp` — MIDI port conflict dialog (commit `ea8744b`)

### 4. `HANDOFF.md` — Known issues updated (commit `d89ff8c`)

Reflects that `__yield` workaround is in place (builds succeed), init patch
placeholder is fixed, and MIDI port conflict now shows a dialog.

### 5. `src/floorBoard.cpp` — Balancer column spacing fix (commit `5032177`)

**Root cause:** `bal1xpos = max(index1, index2) + 1` placed each balancer
one column past the last branch block, creating an empty-column gap. This
gap cascaded through BAL2 and BAL3 formulas, producing the "long gaps" layout.

**Fix:** Removed `+ 1` from all three balancer position formulas and their
cap floor values:
```
Before: bal1xpos = max(index1, index2) + 1
After:  bal1xpos = max(index1, index2)
```
Same for bal2xpos, bal3xpos, and the ordering invariant floors in the
MAX_BAL_SPREAD cap logic.

**Verified:** chain-positions diagnostic confirms all column errors = 0.
Cross-row alignment confirmed (e.g. FX1=FX2=FX3=Comp at same X).
