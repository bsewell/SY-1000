## Session Summary

Compared all 9 INST 1 (Dynamic Synth) tab screenshots against Boss reference images. Fixed layout alignment issues across 7 files, plus a systemic SyComboBox label-width fix.

## Changes Made

### 1. SyComboBox Label Auto-Sizing (`qml/SyComboBox.qml`)
- **Problem**: Fixed 140px `labelWidth` caused labels like "SHAPE", "LOWER", "UPPER" to have huge gaps before dropdown buttons.
- **Fix**: Added a dedicated `TextMetrics` element (`labelMetrics`) to measure label text width independently. `labelWidth` now auto-sizes to `labelMetrics.width + 8` instead of fixed 140px.
- Removed `width` constraint and `elide` from `labelText` — it now renders at natural size.
- `labelWidth` remains overridable (e.g., `labelWidth: 0` in InstrumentPanel still works).

### 2. AltTuneTab (`qml/AltTuneTab.qml`) — Full Rewrite
- **Problem**: Used 8-column grid with controls mixed into pitch row; missing ON/OFF switch; wrong row order (FINE after BEND DEPTH).
- **Fix**: Restructured to match reference:
  - Row 0: ON/OFF (hex3 "00"), STR BEND SW, BEND CONTROL (3 controls)
  - Row 1: PITCH 6-1 (6 knobs)
  - Row 2: FINE 6-1 (6 knobs)
  - Row 3: BEND DEPTH 6-1 (6 knobs)

### 3. FilterTab (`qml/FilterTab.qml`)
- **Problem**: TYPE and SLOPE dropdowns stacked vertically.
- **Fix**: Placed in a `Row` side-by-side to match reference layout.
- Removed unused `comboWidth: 340` property.

### 4. OscTab (`qml/OscTab.qml`)
- **Problem**: Extra SySwitch (hex3 "13") and SyComboBox (hex3 "17") in row 2, not present in reference.
- **Fix**: Removed both. Row 2 now has 6 knobs: PWM ENV ATTACK through P. BEND CTL.
- Removed unused `comboWidth: 340` property and hardcoded `implicitWidth` on WAVEFORM combo.

### 5. Lfo1Tab (`qml/Lfo1Tab.qml`)
- **Problem**: Missing BPM knob in second row (reference shows FADE TIME, SYNC, BPM).
- **Fix**: Added BPM knob at hex2 "12", hex3 "3E" (master tempo, same address as SeqTab).
- Removed hardcoded `implicitWidth: 340` on SHAPE combo.

### 6. Lfo2Tab (`qml/Lfo2Tab.qml`)
- Same changes as Lfo1Tab: added BPM knob, removed hardcoded combo width.

### 7. SeqTab (`qml/SeqTab.qml`)
- Removed hardcoded `implicitWidth: 180` from all three target combos (PITCH TARGET, CUTOFF TARGET, LEVEL TARGET). Now auto-sizes with the rest.
