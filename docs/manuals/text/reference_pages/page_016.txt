Editing the Effects

Editing While Viewing All Parameters

You can edit the effect settings, and change the order of blocks such
as output and send/return (the effect chain).

You can also edit while viewing a list of all parameters for the
selected block.

Basic Operation for Effect Editing

1.	 In the edit screen, long-press the [6] knob.
Shows a list with all parameters of the selected block.

1.	 Press the [EFFECTS] button.
The edit screen (effect chain) appears.

2.	 Turn the [1]–[6] knobs to edit the parameter
settings.
2.	 Turn the [6] knob to select the block that you want
to edit.

MEMO
You can use the [K] [J] (PAGE) buttons to switch between the
parameters.

The selected block is enclosed by a thick frame.

* By pressing the [6] knob you can turn the selected effect on/off.
Effects that are off are shown in gray. When the effect is turned on,
it is shown in white.
