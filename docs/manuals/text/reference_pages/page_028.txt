About STEP CURVE
STEP CURVE provides a choice of the following 13 types.

* If a parameter to be controlled (PITCH, CUTOFF, LEVEL) is not
assigned to the sequencer, these settings have no effect (they are
shown in gray).
* Settings of steps that are beyond the value specified by LENGTH
have no effect (they are shown in gray).
