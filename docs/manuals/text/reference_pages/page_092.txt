FACTORY RESET

MODE SETTING

Initializes the SY-1000 to its factory-set condition.

Here you can specify whether you’re using the SY-1000 with a guitar
connected or with a bass connected.

MEMO
Only the currently selected mode is initialized. (If you execute Factory
Reset while in GUITAR MODE, the settings of BASS MODE are not
initialized.)
